# TransitSpectroscopy-backend-async

Разработка интернет-приложений (РИП) 2025

Определение химического состава планеты методом транзитной спектроскопии

Разработанное приложение позволит астрофизикам-исследователям создавать и формировать заявки на расчёт химического состава атмосферы экзопланет по данным транзитной спектроскопии, а модераторам завершать или отклонять данные заявки.
